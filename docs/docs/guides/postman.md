# Postman Guide

We provide a Postman collection to easily test the API.

## Importing

1. Open Postman.
2. Click "Import".
3. Select the `postman/ShoppingProject.postman_collection.json` file from the repository.

## Environment

Configure a "Local" environment with:
- `baseUrl`: `http://localhost:5000` (Gateway) or `http://localhost:5001` (API)
- `apiKey`: Your configured API key
