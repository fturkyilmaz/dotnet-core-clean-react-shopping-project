# Identity API

Manage user authentication and accounts.

## Endpoints

### Login
`POST /api/v1/identity/login`

### Register
`POST /api/v1/identity/register`

### Refresh Token
`POST /api/v1/identity/refresh-token`

### Reset Password
`POST /api/v1/identity/reset-password`

### Forgot Password
`POST /api/v1/identity/forgot-password`

### Me
`GET /api/v1/identity/me`

### Me Update
`PUT /api/v1/identity/me`

### Roles
`POST /api/v1/identity/roles/User`
